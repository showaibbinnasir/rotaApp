# Universal Rota Apps by Showaib Bin Nasir

Hello guys, i am Showaib. This is one of my project that i have made through javaScript.

You can add any staff you want. Simply fix the starting date.
Then you can drag and drop any staff to the day box. you can add multiple staff in one day. after adding staff
you will have the option to add or change time shifts. after finishing a day you can collapse the window and start work on another day. 
You dont need to be tensed about loosing your data, data will be stored on local session storage so that while you make refresh or even you make exit from the browser
inserted data always be there. Again to remove all data you can see a button call clear all. press on it, all data will be wiped out obviously after getting your conversation.

Its not finished yet, you can actually download rota as table from the system. there will be a button called download pdf. just hit this, you will have the table with days dates shifts names.
Also there is a section called weekly hours summary, here you can see how many hours you staffs gonna working per week. This will also be generated on the same pdf while you click the button.

🧩 Core Features
* Add staff members to the system
* Set or fix the starting date for scheduling

📅 Scheduling System
* Drag and drop staff into specific day boxes
* Assign multiple staff members to a single day
* Add and modify shift times for each staff member
* Organize staff schedules across multiple days

📂 Day Management
* Collapse individual day windows to keep the interface organized
* Work on multiple days separately without losing previous data

💾 Data Persistence
* Automatically saves all data using local session storage
* Data remains even after:
* Page refresh
* Closing and reopening the browser
* Manual option to clear all stored data using a “Clear All” button

🧾 Export & Reporting
* Download rota (schedule) as a PDF
* PDF includes:
* Table of days
* Dates
* Assigned staff
* Shift times

⏱ Weekly Summary
* Displays weekly working hours per staff member
* Automatically calculates total hours per week
* Included in the downloaded PDF report
